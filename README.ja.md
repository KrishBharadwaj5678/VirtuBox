# 📦 バーチュボックス

**バーチュボックス** は、指を使って画面上の**仮想ボックスをドラッグ＆ドロップできる**Pythonベースの仮想インターフェースです。

![バーチュボックス](https://github.com/KrishBharadwaj5678/VirtuBox/blob/main/VirtuBoxDemo.gif)

## 🚀 機能

| 機能                   | 説明                          |
| :------------------- | :-------------------------- |
| 👆 **指によるドラッグ＆ドロップ** | 指を使って仮想ボックスを簡単に操作できます       |
| 🎯 **高精度トラッキング**     | MediaPipe による正確な指の動きを実現     |
| 📦 **仮想ボックス**        | 画面上のボックスをリアルタイムで移動・操作可能     |
| ⚡ **スムーズなパフォーマンス**   | 遅延のないリアルタイム操作を体験できます        |
| 💻 **クロスプラットフォーム対応** | Windows / macOS / Linux で動作 |
| 🔒 **ローカル動作**        | 完全オフラインで動作し、すべてローカル処理       |

---

## 🛠️ 技術スタック

| 技術                | 説明                                |
| :---------------- | :-------------------------------- |
| 🐍 **パイソン 3.x** | アプリケーションのコアプログラミング言語              |
