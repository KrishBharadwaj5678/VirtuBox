# 📦 VirtuBox

**VirtuBox** é uma interface virtual baseada em Python que permite **arrastar e soltar caixas virtuais na tela usando os dedos**.

![VirtuBoxDemo](https://github.com/KrishBharadwaj5678/VirtuBox/blob/main/VirtuBoxDemo.gif)

## 🚀 Funcionalidades

| Funcionalidade                        | Descrição                                                |
| :------------------------------------ | :------------------------------------------------------- |
| 👆 **Arrastar e soltar com os dedos** | Controle caixas virtuais facilmente usando os dedos      |
| 🎯 **Rastreamento preciso**           | Rastreamento preciso dos dedos alimentado pelo MediaPipe |
| 📦 **Caixas virtuais**                | Mova e interaja com caixas na tela em tempo real         |
| ⚡ **Desempenho suave**                | Experiência em tempo real sem travamentos                |
| 💻 **Multiplataforma**                | Funciona perfeitamente no Windows, macOS e Linux         |
| 🔒 **Execução local**                 | 100% offline — todo o processamento é feito localmente   |

---

## 🛠️ Tecnologias Utilizadas

| Tecnologia        | Descrição                                                      |
| :---------------- | :------------------------------------------------------------- |
| 🐍 **Pitão 3.x** | Linguagem principal usada na aplicação                         |
| 📷 **OpenCV**     | Responsável pela visão computacional e processamento de imagem |
| ✋ **MediaPipe**   | Permite rastreamento preciso de mãos e dedos                   |
| 📦 **Zona CV**     | Simplifica a integração entre OpenCV e MediaPipe               |
