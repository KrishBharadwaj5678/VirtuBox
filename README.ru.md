# 📦 ВиртуБокс

**ВиртуБокс** это виртуальный интерфейс на Python, который позволяет **перетаскивать виртуальные блоки на экране с помощью пальцев**.

![ВиртуБокс](https://github.com/KrishBharadwaj5678/VirtuBox/blob/main/VirtuBoxDemo.gif)

## 🚀 Возможности

| Возможность                    | Описание                                                                |
| :----------------------------- | :---------------------------------------------------------------------- |
| 👆 **Перетаскивание пальцами** | Управляйте виртуальными блоками с помощью пальцев                       |
| 🎯 **Точное отслеживание**     | Высокоточное отслеживание движений пальцев с помощью MediaPipe          |
| 📦 **Виртуальные блоки**       | Перемещайте и взаимодействуйте с объектами на экране в реальном времени |
| ⚡ **Плавная работа**           | Быстрая и безлаговая работа в реальном времени                          |
| 💻 **Кроссплатформенность**    | Работает на Windows, macOS и Linux                                      |
| 🔒 **Локальная работа**        | 100% офлайн — вся обработка выполняется локально                        |
